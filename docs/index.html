    <h3>Overview</h3>

    <p>

        This pipeline extracts 50,000 NYC building permit records from the NYC Department of Buildings 

        via the Socrata Open Data API, transforms the raw data using Apache PySpark, and loads the 

        cleaned dataset into a Snowflake cloud data warehouse. The project demonstrates a complete 

        data engineering workflow applied to a real-world AEC dataset.

    </p>
